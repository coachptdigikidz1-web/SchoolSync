#SchoolSync - Landing Page
Deskripsi Singkat
SchoolSync adalah sebuah website landing page sederhana yang dirancang sebagai platform kolaborasi digital antara guru dan murid. Website ini bertujuan untuk mempermudah akses informasi jadwal, tugas, dan komunikasi dalam lingkungan sekolah secara terpusat dan modern.

Fitur Utama
Navigation Bar Responsive: Menu navigasi yang menyesuaikan tampilan di HP maupun Laptop.
Hero Section: Bagian utama yang informatif dengan ajakan aksi (Call to Action).
Feature Cards: Menampilkan keunggulan layanan dalam bentuk kartu yang rapi.
Information Section: Penjelasan lebih detail mengenai keunggulan platform.
Interactive Contact Form: Formulir interaktif yang dapat merespons input pengguna menggunakan JavaScript.
Footer: Bagian informasi hak cipta dan penutup website.
Tools yang Digunakan
HTML5: Untuk membangun struktur dasar website.
CSS3: Untuk desain layout (Flexbox & Grid) serta responsivitas.
JavaScript (Vanilla): Untuk memberikan interaktivitas pada form dan navigasi.
Vercel: Sebagai platform untuk deployment (hosting website).
Visual Studio Code: Sebagai text editor untuk menulis kode.
